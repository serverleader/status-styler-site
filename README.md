## Why people love it

- **Type it once, style it 55 ways.** Live preview of every style as you
  type. Tap any style to copy; paste it anywhere. Styled text is real
  Unicode, so it survives every app, bio, and username field.
- **A keyboard that speaks 55 styles.** Enable the Status Styler keyboard
  and type styled text *directly* in any app ... pick a style, type, done.
  No app-switching, no copy-paste.
- **Find your vibe fast.** Styles are organized into browsable categories
  (Classic, Script, Bubbles, Chaotic, Retro...) with filters and sorting,
  in both the app and the keyboard.
- **Private by design.** No accounts. No ads. No analytics. No network
  access at all. The keyboard never requests Full Access, so it
  physically cannot log what you type. Apple's privacy label:
  *Data Not Collected*.
- **One unlock, yours forever.** Five styles are free. A single one-time
  purchase unlocks all 55 ... plus every style added in the future. No
  subscription, ever.

## A little history

Status Styler first hit the App Store in 2011, back when styled SMS
statuses were the height of fashion. Version 2.0 is a complete
ground-up rebuild for modern iOS: SwiftUI, a custom keyboard extension,
and more than triple the styles of the original.
